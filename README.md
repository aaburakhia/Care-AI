# Care-AI
An integrated health co-pilot using AI.

## Features

### 1. Symptom Checker
AI-powered symptom analysis to help users understand their health concerns.

### 2. MindWell Chatbot
Mental health support chatbot for emotional well-being.

### 3. Diagnostic Imaging
Medical image analysis capabilities.

### 4. Medication Manager (NEW)
Track your medications with smart reminders:
- Add medications with name, dosage, and schedule
- Set reminder preferences (15 minutes, 30 minutes, or 1 hour before dose time)
- View all medications with live status updates
- Get notified when it's time to take your medication
- Edit or delete medications as needed

All medication data is securely stored and synced with your account.
